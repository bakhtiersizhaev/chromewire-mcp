# Short hype name and descriptions

## Name

**ChromeWire MCP**

## English

ChromeWire MCP connects the official Codex Chrome Extension to MCP-compatible AI CLIs, giving agents safe localhost control of real Chrome tabs without exposing cookies, history, or profile secrets by default.

## Русский

ChromeWire MCP соединяет официальный Codex Chrome Extension с AI CLI через MCP и даёт агентам безопасное локальное управление реальными вкладками Chrome без выдачи cookies, истории и приватных данных профиля по умолчанию.

## 中文

ChromeWire MCP 将官方 Codex Chrome Extension 连接到支持 MCP 的 AI CLI，让智能体可以通过本地 localhost 安全控制真实 Chrome 标签页，默认不暴露 cookies、历史记录或浏览器配置隐私。
