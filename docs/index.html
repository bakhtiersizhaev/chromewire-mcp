ChromeWire MCPLocalhost browser bridge for AI agents

Real Chrome. Wired to MCP.

ChromeWire MCP connects the official Codex Chrome Extension to GSD, Codex CLI, Claude Code, Cursor, Windsurf, and other MCP-compatible tools.

GitHub repo Official extension

Install

git clone https://github.com/bakhtiersizhaev/chromewire-mcp.git
cd chromewire-mcp
npm install
npm test
npm run doctor
npm start

Endpoint: http://127.0.0.1:8962/mcp

How it works

Chrome profile
  → Official Codex Chrome Extension
  → local native host / browser pipe
  → ChromeWire MCP bridge
  → MCP HTTP endpoint
  → AI CLI tools
Safe by defaultBound to localhost. No cookies, history, passwords, or storage exposed by default.
Real browserOpens, clicks, types, scrolls, screenshots, and reads visible text in actual Chrome tabs.
Agent-readyIncludes install/debug skills, MCP examples, CI, doctor checks, and smoke tests.

Requirements

  • Node.js 20+
  • Google Chrome
  • Official Codex Chrome Extension
  • Windows for the current working transport; macOS/Linux adapters are planned.
Region note: if the Chrome Web Store listing is unavailable in your region, a USA IP via VPN/proxy may be required to install the official extension. Use only legal, policy-compliant access methods.
